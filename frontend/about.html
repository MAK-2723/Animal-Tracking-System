About Our Project

This project develops an AI-powered, non-invasive animal tracking system using real-time image processing to monitor wildlife and mitigate human-animal conflicts

Project Overview

The Animal Tracking System Using Image Processing aims to develop an automated, non-invasive solution for real-time detection, classification, and tracking of animals in urban and rural environments. Traditional tracking methods, such as radio collars or manual observation, are intrusive, costly, and limited in scalability. This project addresses these gaps by leveraging advanced image processing and machine learning techniques to monitor animal movements efficiently.
